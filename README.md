# BMI-Calculation-Tool
Overview

This program asks the user for their height in inches and weight in pounds, then calculates their Body Mass Index (BMI). It displays the raw BMI value, a formatted f-string version, and a rounded BMI value. This demonstrates basic input handling, arithmetic operations, and formatted output in Python. 

Calculatebmi (1)

Features

User input for height and weight

BMI calculation using the standard formula

Output shown in three formats: raw, f-string, and rounded

Simple and beginner-friendly structure
